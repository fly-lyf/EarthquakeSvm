function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;

         <DIV class=Custom_UnionStyle>
<P>　　2月27日，按照《中国防震减灾百科全书》总编委会组织协调办公室要求，在中国地震局应急指挥大厅召开了《中国防震减灾百科全书震灾应急与救援卷》辞条纲目第五次送审稿专家评审会。《百科全书》总编办吴宁远研究员、民政部救灾司曾伟副处长、总参应急办张博正团职参谋、武警总医院杨炯副主任医师、国家安全生产应急救援指挥中心指挥协调部甄小丰副主任、中国地震局震灾应急救援司赵明司长、中国地震应急搜救中心吴卫民主任、《震灾应急与救援卷》主编徐德诗研究员等16名专家和相关人员参加了会议。赵明司长主持会议，吴卫民主任担任评审专家组组长。</P>
<P>　　首先，《震灾应急与救援卷》主编徐德诗研究员从《震灾应急与救援卷》编撰的背景、特点以及编撰的难度和成果等几方面用PPT的形式对编撰情况作了详细汇报。随后，专家们各抒己见，对《震灾应急与救援卷》辞条纲目的编撰体例、内容、科学性、准确性、重复性等方面的不妥之处提出了自己的意见和建议。评审专家组进行了认真评议，一致认为，《应急与救援卷》辞条纲目框架合理、内容全面、重点突出，体现了地震应急救援工作实际和发展水平，符合百科全书辞条纲要的编写要求，基本覆盖了地震应急救援工作的全部内容。《中国防震减灾百科全书 震灾应急与救援卷》辞条纲目在掌声中通过了专家评审。</P>
<P>　　赵明司长指出，项目组要认真落实评审意见，适时召开一次相关部门会议征询意见，提出将“体现当前地震应急救援准备与处置的新要求”纳入《震灾应急与救援卷》辞条纲目的具体措施；局震灾应急救援司继续积极支持，调动行政资源，推进《震灾应急与救援卷》编撰工作。他还向所有参加、支持《震灾应急与救援卷》辞条纲目编撰工作的单位、专家表示感谢。</P></DIV>
