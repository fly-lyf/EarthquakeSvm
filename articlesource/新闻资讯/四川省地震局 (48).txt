function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);

<P>　　为加强和规范德阳市防震减灾示范社区建设管理，德阳市根据中国地震局《地震安全示范社区管理暂行办法》和《四川省防震减灾示范社区创建管理办法》，结合德阳市实际，拟制定出台《德阳市防震减灾示范社区创建管理办法》，德阳市防震减灾局积极推进制定工作，现已形成征求意见稿。 </P>
<P>　　1月13日，德阳市防震减灾局召开征求意见座谈会，会议由德阳市防震减灾局局长黄声德主持，副局长张洪梅以及监测科、应急科、震防科负责人参加。会议对该《办法》（征求意见稿）相应条款进行了讨论，提出了修改完善意见。黄声德强调：德阳市防震减灾示范社区创建要以最大限度减轻地震灾害损失为宗旨，以提高社区居民的防震减灾意识、增强应急避险能力为目标，力争在今后的创建工作中取得优异成绩。 </P>
<P>　　1月15日，该《办法》（征求意见稿）在德阳市防震减灾局网站上全文公布，公开征求各县（市、区）防震减灾局意见，征求意见结束后，将尽快报市法制办审查。 </P>
