function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);

}

         <P class=Custom_UnionStyle>　　<SPAN>在新春佳节即将到来之际，为使精准扶贫点木里县瓦厂镇纳子店村贫困户感受到党和政府的温暖，过上一个愉快、祥和的春节,1月20日，四川省地震局、四川省农科院相关负责人及驻村工作组成员前往纳子店村开展春节节前慰问活动。 </P>
<P class=Custom_UnionStyle>　　慰问组一行对纳子店村34户精准扶贫建档立卡户进行了走访慰问，为他们每户送去慰问金，并致以新春的问候和祝福。走访中，慰问组仔细询问了解各个贫困户的家庭收入、劳动力等基本情况，听取每户贫困户脱贫想法，分析探讨脱贫方案。鼓励他们振奋精神、坚定信心，与帮扶单位一道同心协力克服困难，抓好精准扶贫机会，搞好相关扶持产业，争取早日脱贫致富。 </P>
<P class=Custom_UnionStyle>　　最后，慰问组对纳子店村1名孤儿及2名贫困学生家庭进行了走访慰问，询问了解学习及生活中面临的困难，鼓励他们要树立信心、努力学习，并祝愿他们在各帮扶单位及社会各界人士的帮助关心下度过一个温暖的春节。&nbsp;</P>
