function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);
	newWindow1.focus();// 窗口获取焦点
}

<P align=left>　　<SPAN>2013</SPAN><SPAN>年<SPAN>6</SPAN>月<SPAN>14</SPAN>日，四川省地震局监测预报处在成都市组织召开<SPAN>2013</SPAN>年全度省地震监测预报工作会议。各市州防震减灾局、省地震局机关各部门、局属各单位负责人，以及省地震局各学科技术管理组负责人等<SPAN>90</SPAN>余人参加了会议，省地震局党组书记、局长张宏卫，党组成员、副局长李广俊出席会议，并作重要讲话。</SPAN></P>
<P align=left>　　<SPAN>这次会议，是在汶川地震五周年、芦山地震两个月之际，我省防震减灾面临重大机遇期的特殊背景下召开的重要会议，会议的主要任务是传达学习全国监测预报工作会议精神，回顾总结上一年度全省监测预报工作，安排部署本年度全省监测预报重点工作。</SPAN></P>
<P align=left>　　<SPAN>张宏卫在讲话中指出，近年来全省地震监测工作作出了勤奋努力，取得显著成效，希望大家在科学总结芦山“<SPAN>4</SPAN></SPAN><SPAN>・</SPAN><SPAN>20</SPAN><SPAN>”<SPAN>7.0</SPAN>级强烈地震基础上，进一步深入思考地震监测预报工作、切实加强震情跟踪、不断提升监测预报能力和水平。</SPAN></P>
<P align=left>　　<SPAN>李广俊在会上作了工作报告，全面回顾总结了全省的监测预报工作并对当前及今后一段时期的全省地震监测预报工作进行了部署。</SPAN></P>
<P>　　<SPAN>省地震局测震</SPAN><SPAN>、电磁、形变、流体、信息、前兆台网运行等</SPAN><SPAN>各学科技术管理组分别作了工作报告；</SPAN><SPAN>康定地震中心站、西昌地震中心站、测绘工程院、攀枝花市防震减灾局、宜宾市防震减灾局等单位分别</SPAN><SPAN>进行了监测运维和质量管理经验交流；省地震局与局属监测单位负责人签订了<SPAN>2013</SPAN>年度地震观测质量目标责任书；会议还对上年度获得全国和全省评比前三名的单位及监测工作先进单位进行了表彰奖励。</SPAN></P>
