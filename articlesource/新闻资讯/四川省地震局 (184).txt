function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);

<P>　　<SPAN id=media_name><SPAN data-sudaclick="media_name"><A href="http://sichuandaily.scol.com.cn/2014/08/30/20140830637334088018.htm" target=_blank _fcksavedurl="http://sichuandaily.scol.com.cn/2014/08/30/20140830637334088018.htm">四川在线-四川日报</A></SPAN></SPAN>本报讯 （记者 王玉姗）8月29日，联合国国际搜索与救援咨询团亚太地震应急演练核心项目在成都举行。演练由中、美两国共同资助，其中8月25日至27日在北京观摩中国国际救援队能力测评复测演练，28日参会的部分搜索与救援团成员前往映秀进行考察，29日进行经验交流研讨会，30日为桌面演练。</P>
<P>　　联合国国际搜索与救援咨询团亚太地震应急演练1到2年举办一次，旨在通过开展程序性的桌面推演，促进各国际救援队、国际人道主义组织、举办国应急响应机构与救援力量对联合国国际搜索与救援咨询团灾害响应系统的认知和熟悉程度，加强各国际人道主义救援力量在灾害现场的沟通协调能力和联合开展救援工作的能力。此次是中美两国在亚太地区联合支持的第3次地震演练，由联合国人道主义事务办公室主办，中国地震局承办，来自中国、美国、日本、澳大利亚、新加坡等16个国家的应急救援专家和救援队员及联合国代表共142名人员参加。</P></DIV>
