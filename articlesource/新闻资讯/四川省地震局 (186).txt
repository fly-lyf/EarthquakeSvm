function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度

<P>　　<IMG style="BORDER-RIGHT-WIDTH: 0px; BORDER-TOP-WIDTH: 0px; BORDER-BOTTOM-WIDTH: 0px; BORDER-LEFT-WIDTH: 0px" src="./W020140830409711808205.jpg" OLDSRC="W020140830409711808205.jpg" _fcksavedurl="/webpic/W0201408/W020140830/W020140830409711808205.jpg"/>　四川新闻网成都8月29日讯(记者&nbsp;何雪)四川新闻网记者从省地震局获悉，2014年8月25日至30日，联合国国际搜索与救援咨询团在我国举行亚太地震应急演练。此次演练是中美两国在亚太地区联合支持的第3次地震演练，演练活动分别在北京和成都举行，其中，成都地区的演练活动主要包括现场考察和桌面演练。今(29)日上午，联合国国际搜索与救援咨询团核心演练活动在成都正式启幕并举行“中国和国际地震应急经验分享”研讨会。　　据悉，此次演练由中美双方共同资助，由联合国人道主义协调办公室(OCHA)下属的现场协调与支持中心(FCSS)和中国地震局联合主办，中国地震应急搜救中心、四川省地震局等单位具体承办，旨在通过演练增强参与各方对国际搜索与救援咨询团(INSARAG)灾害响应系统的认知和熟悉程度，加强各国际人道主义救援力量在灾害现场的沟通协调能力和联合开展救援工作的能力。　　四川省地震局向四川新闻网记者介绍，来自美国、日本、澳大利亚、新加坡等17个国家及联合国代表在内的95名应急救援专家和救援队员参与了演练。其中，中方34名，分别来自中国地震局、四川省地震局、中国国际救援队及四川省救援队。　　据了解，此次演练的核心演练活动将于今、明两日在成都举行。其中，今日为正式演练活动的准备阶段，将以PPT汇报的研讨会模式使参演人员熟悉受灾国的应急管理机制及基于国际搜索与救援咨询团(INSARAG)指南的国际搜救行动的派遣和协调程序。这部分内容的目的是为参演人员提供关于国际搜索与救援咨询团(INSARAG)和灾害评估与协调队伍(UNDAC)程序方法的培训，使其熟悉在后续演练中将要用到的地震灾区相关情况，也为不同组织的相关人员提供一个相互熟悉了解的平台和机会。</P></DIV>
