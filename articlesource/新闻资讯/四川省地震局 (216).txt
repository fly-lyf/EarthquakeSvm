function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);

}

         <P class=Custom_UnionStyle>　　<SPAN>在<SPAN>"5</SPAN>・<SPAN>12"</SPAN>汶川地震纪念日和第七个“防灾减灾日”到来之际，“平安中国”防灾宣导千城大行动德阳站启动仪式于<SPAN>5</SPAN>月<SPAN>8</SPAN>日在四川省德阳市东汽八一中学举行。四川省地震局副局长雷建成，德阳市人大副主任胡宗林，德阳市副市长、市防震减灾领导小组组长王箭，德阳市政协副主任吴玉华等领导及市防震减灾局、市教育局、市政府应急办、市民政局等部门负责人出席了启动仪式。东汽八一中学<SPAN>3000</SPAN>余教职员工和学生、德阳市各大专院校、直属学校有关负责人参与活动。</SPAN></P>
<P class=Custom_UnionStyle>　　<SPAN>仪式上，德阳市防震减灾局局长黄声德向东汽八一中学赠送了《防震减灾中小学教育读本》<SPAN>3000</SPAN>册以及防灾宣传品；学校师生进行了应急疏散演练以及心肺复苏、伤口包扎、感恩文艺汇报表演；各级领导和嘉宾共同点亮了<SPAN>2015</SPAN>年“平安中国”防灾宣导千城大行动水晶球，并在“防震减灾进校园”签名墙上签名。晚上<SPAN>8</SPAN>时组织全校师生共同观看防震减灾主题影片《<SPAN>5</SPAN>・<SPAN>12</SPAN>汶川不相信眼泪》。</SPAN></P>
<P class=Custom_UnionStyle>　　<SPAN>根据平安中国防灾宣导组委会要求，德阳市还分别在旌阳区孝泉镇孝泉中学、旌阳区杨嘉镇千秋中学、德阳市庐山路小学、旌阳区南塔社区、经开区旌湖社区公益放映防震减灾专题动画片和防灾文化电影活动，以提升公众防震减灾意识和自救互救能力。</SPAN></P>
