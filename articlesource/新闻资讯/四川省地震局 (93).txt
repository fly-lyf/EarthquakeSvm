function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);

<P align=left>　　<SPAN>12</SPAN><SPAN>月<SPAN>2</SPAN>日<SPAN>-4</SPAN>日，四川省地震局在眉山市举行<SPAN>2015</SPAN>年度全省地震系统政务工作培训。省地震局副局长吕志勇出席培训班并讲话。四川省地震局机关各部门、局属各单位和各市（州）防震减灾局（办）的部分领导及办公室工作人员参加培训。</SPAN></P>
<P align=left>　　<SPAN>此次培训邀请到四川省政府办公厅、四川省保密局、成都卓瑞特公司的专家，分别就公文办理、保密工作、四川省地震局<SPAN>OA</SPAN>办公系统等方面的内容，向参训人员授课。政务信息和政务工作的先进单位的参训代表在经验交流环节中发表主题演讲。最后，省地震局办公室负责人对<SPAN>2015</SPAN>年全省地震系统政务工作进行了总结，并就下一步工作提出了要求和计划。</SPAN></P>
<P align=left>　　<SPAN>吕志勇在此次闭幕式上讲话，传达了全国<SPAN>2016</SPAN>年度地震趋势</SPAN><SPAN>会商会精神。简要回顾了<SPAN>2015</SPAN>年政务工作，指出全省地震系统<SPAN>2015</SPAN>年政务工作成效显著，表现在政务工作更加规范、综合协调能力增强、督查督办重点突出、新闻宣传力度加大等方面。关于今后政务工作，指出一是要进一步认识政务工作和保密工作的重要性；二是要进一步提高政务运转效率；三是要进一步强化督促落实；四是要进一步主动加强宣传。</SPAN></P>
