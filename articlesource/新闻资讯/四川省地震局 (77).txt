function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;

<P>　　1月25日，四川省地震局召开党组中心组扩大学习会，传达学习2016年全国地震局长会暨党风廉政建设工作会议精神。党组书记、局长张宏卫主持会议，局领导吕弋培、李广俊、雷建成、李明、王力出席会议，机关各部门、局属各单位负责人，地震应急保障中心、成都地震基准台主要负责人参加会议。 </P>
<P>　　张宏卫在会上传达了全国地震局长会暨党风廉政建设工作会议情况，从回顾2015年防震减灾工作新进展和十二五期间防震减灾事业发展，谋划“十三五”期间防震减灾事业发展思路和举措，部署2016年防震减灾和党风廉政建设重点工作任务等方面，传达学习了中国地震局局长陈建民在会上作的主题报告。 </P>
<P>　　会议对全面贯彻落实全国地震局长会暨党风廉政建设工作部署提出了具体要求：一是各部门、各单位要组织干部职工进一步传达学习会议精神，进一步转变观念，切实把思想和行动统一到中央、中国地震局和局党组的要求上来，重点把握好当前防震减灾工作的新思路、新任务、新要求；二是结合本部门、本单位实际，制定好2016年工作计划及任务方案，建立健全各项保障机制，研究部署贯彻落实措施；三是切实加强党风廉政建设，落实“两个主体”和“一岗双责”，开展好各项专项检查，将今年作为“管理年”，有关部门要拿出具体实施方案，严管干部队伍，严肃党内生活，严正作风建设，严守廉政底线；四是切实做好监测预报工作，坚持“震情第一”意识，强化监视跟踪，做好应急准备。 </P>
<P>　　会议同时对近期切实加强应急值守、安全防范等工作做出了安排部署。</P>
