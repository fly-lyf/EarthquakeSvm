function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);

}

<P align=left>　　<SPAN>11</SPAN><SPAN>月<SPAN>17</SPAN>日，凉山州人民政府办公室印发《关于加强应急避难场所建设的意见》（凉府办发〔<SPAN>2012</SPAN>〕<SPAN>68</SPAN>号），进一步推进全州应急避难场所建设工作。</SPAN></P>
<P align=left>　　<SPAN>文件要求到<SPAN>2012</SPAN>年底，凉山州政府相关部门要完成应急避难场所建设规划的编制及相关政策措施和标准、规范的制定；各县市完成应急避难场所的规划。到<SPAN>2013</SPAN>年底，西昌城区建成<SPAN>2-3</SPAN>个<SPAN>I</SPAN>类应急避难场所，县城建成<SPAN>1-2</SPAN>个<SPAN>II</SPAN>类应急避难场所。到<SPAN>2015</SPAN>年底，全州<SPAN>30%</SPAN>的集镇建设<SPAN>III</SPAN>类应急避难场所，在<SPAN>2016</SPAN>至<SPAN>2020</SPAN>年期间，基本形成以西昌城区、县城、重点集镇为主，远近结合、覆盖面广、设施配套的城镇应急避难场所体系。</SPAN></P>
<P align=left>　　<SPAN>此外，意见还明确了应急避难场所建设的建设标准、安全要求及功能设备，并对今后全州应急避难场所建设的重点工作作出五点要求：一是狠抓调研研究；二是编制专项规划；三是严格建设标准；四是进行统一标识；五是建立信息系统。要求凉山州政府相关部门、各县市人民政府在加强责任落实、加大投入保障、加强部门合作、加强督促检查的前提下，扎实推进全州应急避难场所科学规划和建设，切实提高全州综合防灾减灾能力，最大限度地减少突发性灾害事故造成的人员伤亡和财产损失，促进凉山州经济社会协调可持续发展。</SPAN></P></DIV>
