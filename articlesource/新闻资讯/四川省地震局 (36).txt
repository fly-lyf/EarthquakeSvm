function showDialogSize(url,width,height){
	var availHeight = window.screen.availHeight;
	var availWidth = window.screen.availWidth;
	// 大致居中
	var iWidth = width - 10; // 窗口宽度
	var iHeight = height - 30;// 窗口高度
	var iLeft = (availWidth - width) / 2;
	var iTop = (availHeight - height) / 2;
	var newWindow1 = window.open(url, "","scrollbars=yes,toolbar=no,location=no,Direction=no,resizable=no,Width="+ iWidth + ",Height=" + iHeight + ",top=" + iTop + ",left="+ iLeft);
	newWindow1.focus();// 窗口获取焦点
}

<P>　　<SPAN>扬子晚网讯<SPAN>(</SPAN>记者 王S<SPAN>) </SPAN>紧急地震，跑还是不跑？遇到拥挤踩踏，如何自救<SPAN>……</SPAN>在<SPAN>“5.12”</SPAN>防震减灾日来临之际，<SPAN>“</SPAN>江苏应急公益行<SPAN>”5<SPAN><SPAN>月10</SPAN></SPAN><SPAN><SPAN>日</SPAN></SPAN><SPAN><SPAN>上午</SPAN></SPAN><SPAN>在南京正式启动，还成立江苏首支应急志愿者服务队。据江苏省应急管理办公室主任王平介绍，本次活动将历时一年，每月举行一次，南京为首发站，此后将陆续走进全省13</SPAN></SPAN>个市的社区、企业、学校，旨在唤起<SPAN>“</SPAN>全民学习自救互救、人人懂得应急避险<SPAN>”</SPAN>的意识。</SPAN></P>
<P>　　<SPAN>扬子晚报记者注意到，在今天上午的互动环节，对于应急专家提出的如何应对高楼逃生、地震避险、人群踩踏等问题，很多市民并不知道正确的方式。<SPAN>“</SPAN>市民对实在管用的应急知识的掌握还要加强，特别是要提高避险逃生、自救互救的技能。<SPAN>”</SPAN>王平告诉记者，在接下来的公益性活动中，将通过多种多样的方式，把应急救援的知道送到居民身边，提高公众日常应急安全知识以及应对突发事件的能力。</SPAN></P>
<P>　　<SPAN>此外，江苏首支应急支援服务队也在今天成立，共<SPAN>40</SPAN>名成员。他们中有各类应急救援知识的专业人士，也有热心公益应急事业的社会人士和在校大学生。据介绍，以后，他们将在传播应急救援知识、推动公众应急防灾意识普及方面发挥作用。</SPAN></P></DIV>
